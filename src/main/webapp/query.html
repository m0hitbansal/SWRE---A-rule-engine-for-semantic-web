Query

Query page takes enables different queries to be run on the knowledge base. The input query can also contain literal in the subject/object. For adding a literal in the query, select the Option in the drop down menu and try the required literal in the text box under single quotes, example, 'MT2019001'. Object properties, data properties and sub-properties can also be queried in order to achieve the results. Complexities in the queries can be added by combining multiple triples and connecting them using AND/OR connector.

Backward chaining Query

Backward chaining recursively searches for the result for the current single level query. Backward chaining works only on the literals i.e. only literals should be provided in the subject/object part.
Use the drop down menu to toggle Other value and insert the literals in a single quote. Backward chaining gives a boolean alert based solution to the query.
