University Ontology

A sample ontology built on IIIT-B data

University Ontology tries to capture a basic structure of information about a university.There are 8 classes ,12 object properties and individuals (instances of class).Some of the object properties incorporates characteristics such as Reflexive,Symmetric,Transitive and inverseOf. In addition to this, there are also Data Properties pertaining to each individual that stores more information about that individual.
